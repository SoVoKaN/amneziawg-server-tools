# Quick start

This guide will help you create an awg interface and add a client to it.

## Create Interface
Run amneziawg-manager:

```sh
./manager.sh
```

You will see the menu. 

In the menu, select "Manage Interfaces" and then select "Create interface".

Follow its interactive prompts and answer the questions.

You can choose default option, which will be shown in [brackets] &ndash; press Enter to accept it.

After all actions, the interface will be ready.

## Create client

Run amneziawg-manager:

```sh
./manager.sh
```

In the menu, select "Manage Clients".

Enter the interface name for question.

Then select "Create Client".

Follow its interactive prompts and answer the questions.

You can choose default option, which will be shown in [brackets] &ndash; press Enter to accept it.

After all actions, the client will be ready.
